# MAC and IP Finder

A simple, lightweight utility to quickly find your current IPv4 and MAC addresses. This project includes scripts for both **Python** and **PowerShell**, along with a convenient Windows batch file for quick execution.

## 🚀 Features

-   **Multi-language Support**: Includes implementations in Python and PowerShell.
-   **Windows Optimized**: Comes with a `.bat` file for double-click execution.
-   **No External Dependencies**: Uses built-in libraries for maximum portability.

## 📂 Project Structure

-   `finder.py`: The main Python script.
-   `finder.ps1`: The PowerShell alternative.
-   `run.bat`: A quick-start batch file for the Python script.

## 🛠️ Usage

### 🐍 Python (Standard)
1. Ensure you have Python installed.
2. Run:
   ```bash
   python finder.py
   ```
   *Or just double-click `run.bat`.*

### 🐚 PowerShell (Native Windows)
1. Right-click `finder.ps1` and select **Run with PowerShell**, or use:
   ```powershell
   powershell -ExecutionPolicy Bypass -File finder.ps1
   ```
